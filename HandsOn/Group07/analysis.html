This dataset provides information about the public lighting system in the town of Belmez. The dataset contains the following columns:

  • **ID**: Unique identifier for each streetlight.
  • **Location**: GPS coordinates for the streetlight.
  • **Type**: Type of light used (e.g., LED, sodium, etc.).
  • **Status**: Indicates whether the streetlight is functional or needs maintenance.
  • **Power Consumption**: The power usage of each streetlight in watts.

Data Analysis

The dataset includes approximately 1,000 records, with power consumption values ranging from 50W to 200W. The data is structured in a tabular format and represents real-world entities, such as streetlights and their locations. This makes it easy to transform the data into RDF and link it with other city infrastructure datasets, such as traffic lights or public transportation systems.

License Analysis

The dataset is published under an open license, allowing its use for research and public projects. The license permits the transformation of the data into RDF and its publication on the web. The dataset is provided by the city council of Belmez, which holds the rights to the dataset.

Resource Naming Strategy

For the transformation into Linked Data, the following strategy for naming resources (URIs) has been defined:

  • **URI Form**: The URIs will follow the structure http://example.org/city/lighting/{id}, where `{id}` represents the unique identifier for each streetlight.
  • **Content Negotiation**: Resources will be accessible through different formats (RDF/XML, Turtle, JSON-LD) based on the content negotiation header sent in the HTTP request.
  • **Domain**: The base domain for the resources will be http://example.org.
  • **Path**: The path will follow the pattern `/city/lighting/{id}`, with a specific ID for each streetlight.
