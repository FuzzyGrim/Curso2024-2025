Dataset Requirements Analysis

Dataset Information:

Dataset: Alumbrado Público - Belmez

Source: Alumbrado Público Dataset

Requirement R1: Data in Smart City Domains

The dataset belongs to the Smart City domain as it contains information about public lighting in urban areas, which is a key component of city infrastructure and energy management.

Requirement R2: Data Available as CSV

The dataset is available in CSV format. You can find the file in the /csv directory of this repository.

Requirement R3: Open License

The dataset is published under an open license, allowing it to be transformed into RDF and published online.

Requirement R4: Linkable with Real-world Entities

The dataset can be easily linked with real-world entities like locations. The dataset includes coordinates for each public lighting point, which can be linked with geographic information systems (GIS) for further integration.

Optional Requirements:

Requirement R5: Documentation

There is basic documentation provided with the dataset, which is sufficient for understanding the data structure and its use cases.

Requirement R6: Multiple Data Sources

The dataset is from a single source. However, it can be enriched with additional datasets, such as weather or traffic information, to provide more insights.
