Analysis

Dataset

For the analysis of the dataset we will display the different properties on a table, with their name, type, value range and any additional notes.

horarios.csv

The dataset contains readings for each of the control stations, each line has a different magnitude and/or control station that will have up to 24 different values, one for each hour of the day, depending on when this file was downloaded from the source.
Name Type Value Range Notes
PROVINCIA Integer 28 Code of Madrid province.
MUNICIPIO Integer 79 Code of Madrid municipality.
ESTACION Integer 4-61 Control station code.
MAGNITUD Integer 1-36 Represents specific reading identifier.
PUNTO_MUESTREO String Combination of station, magnitude, and technique. Unique.
ANO Integer 2000-2024 Year of the data collection.
MES Integer 1-12 Month of the data collection.
DIA Integer 1-31 Day of the data collection.
H01, H02, ... Float Various Hourly value. Nullable if reading failure.
V01, V02, ... Character V, N Validation code (V = Valid, N = Not valid).

informacion_estaciones_red_calidad_aire.csv

The dataset contains information about each of the control stations, each line has a different control station with information like its location, id, etc. CODIGO_CORTO is the same as "horario.csv" ESTACION.
Name Type Value Range Notes
CODIGO Integer 28,079,004 — 28,079,061 Unique control station identifier.
CODIGO_CORTO Integer 4-61 Short code representing the station.
ESTACION String Name of the station.
DIRECCION String Address of the station.
LONGITUD_ETRS89 String Longitude.
LATITUD_ETRS89 String Latitude.
ALTITUD Integer 600-730 Altitude of the station location.
COD_TIPO String UT, UF, S Type of station (Urbana Tráfico, Urbana Fondo, Suburbana).
NOM_TIPO String COD_TIPO in detail.
NO2, SO2, CO, PM10, PM2_5, O3, BTX String X or " " Represents whether data is available (X).
COD_VIA Integer 0-32,000,000 Unique code for street or avenue.
VIA_CLASE, VIA_PAR, VIA_NOMBRE String Address details.
Fecha alta Date 01/05/1998 - 02/07/2010 Starting date of the station.
COORDENADA_X_ETRS89, COORDENADA_Y_ETRS89 Float Various Coordinates in ETRS89 system.
LONGITUD, LATITUD Float Various Longitude and latitude.
